# 飞象老师电路题 Demo

一个纯前端 MVP：把电路题图片转成可交互的 HTML + SVG 电路实验。

## 当前能力

- 优先适配 `public/题目1.jpg`
- 优先适配 `public/题目2.jpg`
- 优先适配 `public/题目3.png`
- 支持：
  - 开关交互
  - 滑动变阻器调参
  - 电压/电流实时变化
  - AI 讲解面板
  - 导出实验快照 HTML

## 本地运行

### 方式 1：直接打开

直接双击 `index.html` 即可。

### 方式 2：启动本地静态服务

在项目根目录执行：

```powershell
python -m http.server 8080
```

然后访问：

```text
http://localhost:8080
```

## Demo 说明

- 上传图片后会优先按文件名推断 3 种题型之一。
- 当前版本重点是 **演示效果和题型闭环**，后续可把图片识别替换为真实多模态模型输出。
- 中间层 schema 已经单独保留，方便后期扩展通用性。
